# chicken-fry



ingredents
{
1) ginger garlic paste.
2) turmeric powder.
3) red chili powder.
4) salt.
5) chicken fry masala.
6) cornflower powder.
}

steps to follow [only2]
{
1) mix all the ingredients with the chicken and keep it asside 
   for 1hour

2) after 1hour 
take the chicken and deep fry the chicken in a fry pan with oil 
}
